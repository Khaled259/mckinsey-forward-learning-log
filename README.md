### How to set this up

1.  Go to [GitHub.com](https://github.com) and sign in.
2.  Click the **+** icon in the top right and select **New repository**.
3.  Name it `mckinsey-forward-journal`.
4.  **CRITICAL:** Select **Private**.
5.  Check "Add a README file".
6.  Click **Create repository**.
7.  Use the "Add file" -> "Create new file" button to create the files listed above (`02-Adaptability.md`, etc.) and paste the content in.
